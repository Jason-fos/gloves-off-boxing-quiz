Gloves Off

Gloves off is an online boxing based quiz game for boxing fans, you have 10 minutes to answer 20 questions and only true fans will make it to the end!

Rules and Scoring

  • Create your username
  • You have 10mins to answer 20 questions
  • Each correct answer will gain you 10pts
  • Every wrong answer will deduct 5pts
  • If you're points reach 0 or the timer runs out it's game over
