Gloves Off

Gloves off is an online boxing based quiz game for boxing fans, there are three diffuculties to choose from: Amatuer, Pro and an Undisputed (hard) difficulty for the true fanatics!

Rules and Scoring

  • Create your username
  • Choose you're difficulty level and hit the start quiz button
  • You have 10mins to answer 10 questions
  • Each correct answer will gain you 10pts
  • Every wrong answer will deduct 5pts
  • If you're points reach 0 or the timer runs out it's game over
